# Dickinson Body & Balance

One-page website for a massage therapy business based in Brighton & Hove.

## What it is

A clean, calming landing page designed to convert visitors into bookings. Built to feel professional but personal - exactly what you want for a wellness business.

## Features

- **Full-screen hero** with parallax background image
- **What to Expect section** - walks first-time clients through the process (Book → Arrive → Relax → Feel Amazing)
- **Treatment cards** with hover effects and background images
- **Credentials badges** - Fully Qualified, Fully Insured for trust
- **Mobile-first responsive** with hamburger menu
- **Smooth page loader** animation
- **Subtle paper texture** overlay for warmth
- All CTAs link directly to Instagram for bookings

## Tech

Pure HTML, CSS, and vanilla JS. No frameworks, no build step. Just works.
